# MBA Study Hub

Exam-prep website for the BITS Pilani WILP MBA programme, covering four core courses for Semester 2 (2025-26):

- **Business Statistics** (MBA CC ZG501)
- **Managerial Accounting** (MBA CC ZG502)
- **Managing People and Organizations** (MBA CC ZG511)
- **Marketing Management** (MBA CC ZG503)

Every course has a hub page plus one detailed page per syllabus topic, each with:

- A plain-language explainer
- Key concept bullets
- An interactive multiple-choice quiz
- Expandable short-answer Q&A
- A 150-word answer suitable for written exams
- Topic-wise tips on past question papers

## Live site

Hosted on GitHub Pages — enable Pages in repo settings, then the URL will be:

```
https://kritiagr.github.io/MBA/
```

## Build

The site is generated by a small Python script that scans the local course folders.
The source is in `Websites/_generator/` of the local working copy (not pushed). To
rebuild the web copy that lives in this repo, run from a Windows shell:

```
cd "C:\Users\Vijay Gupta\OneDrive\Documents\MBA\Websites"
publish.cmd
```

That command rebuilds the public site and copies it into `_publish/`.
You then `git add . && git commit && git push` from `_publish/` to update GitHub.

## Local file links

When the site is viewed directly from the OneDrive folder it includes clickable
`file:///` links to every handout, slide, recording and past paper. Those links
are intentionally stripped from the web build for portability and privacy.
